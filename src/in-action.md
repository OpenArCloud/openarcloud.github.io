# OSCP in Action

<a href="https://github.com/OpenArCloud/SpatialDDS-spec" target="_blank" class="project-card">

<p><strong>January 2026</strong></p>
</a>

<a href="/oscp/ubicomp/" class="project-card">
<h3>UbiComp ISWC 2025 Tutorial</h3>

<p><strong>October 2025</strong></p>
</a>

<a href="https://github.com/OpenArCloud/openvps" target="_blank" class="project-card">
<h3>OpenVPS</h3>
<p>Open Visual Positioning System — camera-based localization for AR applications, providing open source tools for visual positioning and mapping.</p>
<p><strong>May 2025</strong></p>
</a>

<a href="/oscp/w3c/" class="project-card">
<h3>W3C Immersive Web Demo</h3>
<p>GeoPose & POI interoperability demos at the W3C Immersive Web Face-to-Face event on Google Campus, Mountain View.</p>
<p><strong>March 2025</strong></p>
</a>

<a href="https://www.auroraviewer.org/" target="_blank" class="project-card">
<h3>Aurora Viewer</h3>
<p>NGI Atlantic–funded research project advancing OSCP toward multi-testbed deployment, including deployment on the COSMOS 5G testbed. Delivers a Unity reference client plus key OSCP improvements.</p>
<p><strong>2022</strong></p>
</a>

<a href="https://www.oscar4us.eu/" target="_blank" class="project-card">
<h3>OSCAR4US</h3>
<p>NGI Search–funded project pushing OSCP toward city-scale location-based AR via OpenVPS, spatial discovery, and a Reality Modeling Language for standardized environment descriptions.</p>
<p><strong>2022</strong></p>
</a>
